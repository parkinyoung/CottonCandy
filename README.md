## 데이터 파일에서 각 레코드를 데이터베이스에 저장하는 API 개발
 1) opencsv 라이브러리를 이용하여 csv파일 한줄을 읽어옴
 2) 정제하지 않은 프로그램정보를 프로그램 테이블에 저장
 3) 테마명 ',' / 지역명 ' '으로 split 후, 테마/지역 저장
 4) 테마와 프로그램의 다대다 관계생성
 5) 테마명 / 프로그램소개 / 프로그램 상세소개를 형태소 분석기를 통해 분해 함
 6) 분석기를 통해 발견된 키워드를 키워드 테이블에 저장
 7) 형태소별 갯수, 타입과 함께 키워드와 프로그램의 다대다 관계생성

## 생태관광정보 데이터 조회, 추가, 수정 API
 1) 조회 : 입력받은 지역명으로 시작하는 지역코드 전달
 2) 수정 : 수정으로 인한 변경된 관계(프로그램-지역, 프로그램-테마, 프로그램-키워드) 재정의시, 효율적인 수행을 위해서 중복된 키워드 목록은 처리 대상에서 제외함 / 관계 추가 대상이 되는 키워드는 추가 작업, 관계 삭제 대상이 되는 키워드는 삭제 작업 진행
 3) 추가 : CSV 데이터 저장시 수행했던 과정 동일 동작  
 
## 프로그램 소개 컬럼에서 특정 문자열이 포함된 데이터의 지역 갯수 출력 API
1) 파라미터로 컬럼 타입과 키워드를 받아 특정 컬럼이 아닌 컬럼으로도 조회 할 수 있도록 구현
2) 파라미터로 들어온 키워드를 형태소분석기를 통해 형태소 분석을 함
3) 형태소분석을 통해 나온 단어들과 일치하는 키워드를 모두 조회
4) 키워드,프로그램 관계를 이용하여 지역 데이터를 정제

## 프로그램 상세 정보를 읽어와서 입력 단어의 출연빈도수 계산 API
1) 파라미터로 들어온 키워드를 형태소분석기를 통해 형태소 분석을 함
2) 형태소분석을 통해 나온 단어들과 일치하는 키워드를 모두 조회
3) 프로그램&키워드 테이블의 카운트를 SUM해서 제공

## (선택) 지역명과 관광 키워드를 입력받아 프로그램 코드를 출력하는 API 
1) 파라미터 지역명으로 시작하는 지역들이 가지는 프로그램 목록 조회
2) 프로그램&키워드 테이블을 이용해 프로그램 필드별 키워드 포함 수를 조회
3) 각 필드 별 (가중치 * 수)의 합을 프로그램의 추천값으로 결정 후, max값을 가진 프로그램 전달
